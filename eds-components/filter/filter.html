Component: Filter | Variant: Filter with static content
How it works:
  • By default, all static content is displayed.
  • When user selects a filter option, the content updates to reflect the selected criteria.
  • User can clear the selected filter to reset the content.
  • User can apply multiple filters at once.
  • Selected filters are displayed above the content area.
  • User can unselect a filter either by unchecking on the filter checkbox or by clicking on the "x" icon next to the filter label.
  • On mobile screens, the filter options are collapsed by default and can be expanded by tapping on the filter button.
