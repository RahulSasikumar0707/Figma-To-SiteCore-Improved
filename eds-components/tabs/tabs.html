Component: Tabs | Variant: Tabs base component
  • 'eds-tabs' class is mandatory for all tabs components.
  • Below structure is the base for all variants.
  • Sitecore should be able to add/remove classes on parent level eds-tabs div
  • Make sure to provide unique ids for each tabs component in a page to avoid any conflicts.
  • Sitecore should be able to add img tag, picture tag or svg icon as per requirement inside nav-item along with text.
  • Sitecore should able to add/remove classes like font, color, etc. on nav-link button
  • Make sure to add 'active' class to the default active tab button i.e; the first tab button inside nav-tabs in below example.
  • The first tab button in each tabs component is set as active by default. You can change the active tab by updating the classes and attributes in the DOM.
  • Make sure to add 'aria-selected="true"' to the default active tab button i.e; the first tab button in below example.
  • Sitecore should be able to author values for aria-controls as per the tab pane id.
  • The id attribute value for button and tab pane aria-labelledby attribute value should match and be unique for each tabs component in a page to avoid any conflicts.
  • data-bs-target value should match with the tab pane id value.
  • The tab-pane id here should match with the aria-controls values of the corresponding tab buttons.
  • The tab-pane aria-labelledby values here should match with the id values of the corresponding tab buttons.
  • Make sure to add 'show active' classes to the default active tab pane i.e; the tab content inside first tab pane
